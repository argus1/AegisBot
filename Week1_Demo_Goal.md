# Week-1 Minimal Demo Goal

## Goal Statement
Deliver one repeatable end-to-end compound-task run by Friday EOD:

1. Robot follows line segment to trigger zone.
2. Trigger is detected (AprilTag or color marker).
3. Manipulator executes grasp sequence and secures object.
4. Robot resumes line following while holding object.

## Success Criteria (Pass/Fail)
- [ ] At least 1 full successful run without emergency stop.
- [ ] Line-follow phase completes before and after grasp event.
- [ ] Gripper retains object through resume phase.
- [ ] Run generates telemetry logs for tracking error and task timing.

## Stretch Goal
- [ ] Achieve 3 successful runs in 10 attempts.
